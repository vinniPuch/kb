Отслеживание новых файлов \ состояние файлов

Проверить состояние файлов git status
Добавить файл под контроль версий git add <file/folder>
После внесения изминений в файл который находиться под контролем версий для того чтобы он попал в следующий коммит необходимо добавить его в список stage файлов использовав команду git add git add <file/folder>
Если в файл из случая описанного в предыдущем пунке еще раз внести изминения до коммита то в коммит попадет предидущий слепок а новые правки будут в файле доступны в виде файла который опять нужно добавить в stage как описано выше.  git add <file/folder>
